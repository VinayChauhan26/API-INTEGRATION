News API Integration Web App-
This project is a simple yet effective News Application that I built using HTML, CSS, and JavaScript, with the help of the News API. The idea behind the app was to create a lightweight portal where users can quickly fetch and read the latest headlines from different sources, all inside a clean and responsive interface.

Why I Built This-
I wanted to practice API integration and DOM manipulation in a real-world scenario. News is something everyone consumes daily, so building a news app felt like the perfect way to combine practical coding with a useful outcome. The project also gave me a chance to experiment with styling, layouts, and interactive features that make the app look polished and professional.

How It Works-
The app uses the fetch() method in JavaScript to call the News API. When the user clicks the Fetch News button, the app sends a request to the API and retrieves articles in JSON format. From there, I parse the data and dynamically create news cards that display the headline, description, source, and an image if available. Each card also has a Read More button that links directly to the full article.

The structure is handled with HTML, while CSS takes care of the design. I used bold colors for the background and buttons to make the interface stand out. The layout is responsive, so the cards adjust neatly whether you’re viewing on a desktop or mobile screen.

Features-
Fetches live news articles from multiple sources.

Displays articles in a card-based layout with images and descriptions.

Includes a “Read More” button for direct access to the full story.

Responsive design that adapts to different screen sizes.

Handles missing data gracefully (e.g., no description or image).

Future Plans-
I plan to add category filters (like Technology, Sports, Entertainment), a search bar for custom queries, and maybe even a dark mode toggle. Another idea is to allow users to save favorite articles locally so they can revisit them later. I also want to polish the UI further with animated gradients and transitions to give it a premium feel.

Conclusion
This project was a great learning experience in combining front-end development with API integration. It shows how a simple stack of HTML, CSS, and JavaScript can be used to build something practical and engaging. I’ll continue improving it, but even in its current form, it’s a solid example of how to bring real-time data into a web app.
